# 🎉 Bingo Mixer

**Break the ice. Make connections. Win at fun!**

Bingo Mixer is a playful social bingo game for in-person mixers, meetups, and team events. Find people who match the prompts, mark your board, and race to get 5 in a row!

## ✨ Features

- Icebreaker bingo for any group size
- Customizable questions and themes
- Play in-person, on a big screen, or hybrid
- Modern, responsive UI (React + Vite)
- Ready for hackathons, workshops, and socials

---

## 🛠️ Quick Start

1. **Install dependencies:**
	 ```bash
	 npm install
	 ```
2. **Start the app:**
	 ```bash
	 npm run dev
	 ```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.

---

## 🧑‍💻 Dev Container & Codespaces

- **GitHub Codespaces:** Create a Codespace from your repo—devcontainer is ready to go!
- **VS Code Dev Containers:** Clone locally, then run `Dev Containers: Reopen in Container`.
- **GitHub Pages:** Deploys automatically on push to `main`.
